# JavaScript Basic Works

A personal scratchpad for learning JavaScript fundamentals — written alongside study sessions, with inline notes and experiments directly in the code.

---

## Files

### `JS_OOP_drafts.js`
Covers JavaScript Object-Oriented Programming concepts explored during study:
- Object creation via literals and constructors (`new Object()`, `new Array()`, etc.)
- `Object.defineProperty` — controlling property descriptors (`writable`, `enumerable`, `configurable`)
- Prototype chain and inheritance (`Baby.prototype = human`)
- Property enumeration (`for...in`, `hasOwnProperty`, `propertyIsEnumerable`)
- ES6 `class` syntax and constructor pattern
- Object extensibility (`Object.isExtensible`, `Object.preventExtensions`)
- Getters and setters

Comments are in Turkish — this file was used as an active study log.

---

### `js_basics.html`
Early experiments with variables, functions, and control flow — structured as an HTML page with embedded JavaScript:
- Variable declarations and basic arithmetic
- Function definition (hypotenuse example using `Math.sqrt`)
- `switch` statement skeleton (LoTR character example — incomplete)
- Loop placeholders

---

### `JS_H5_array.js`
Study notes on JavaScript arrays, written during an active session:
- Array creation: literal `[]`, `new Array()`, sized and pre-filled forms
- Element access and assignment via index operator
- `split` / `join` for string-to-array conversion
- `delete` operator vs `splice` for element removal
- Associative arrays (string-keyed) and `for...in` enumeration
- Multi-dimensional arrays (nested `new Array()` initialisation)
- Array methods: `sort`, `concat`, `slice`, `shift`, `unshift`, `pop`, `push`

Comments and notes are in Turkish or English.

---

This is not a polished project; it is a raw learning log.
